# Docker

## Create a docker file

```
FROM - Base image
ARG - Arguments
LABEL - Container reference
COPY - Copy some folder/file.
RUN - Run some install/modify command
EXPOSE - Container port
ENTRYPOINT - Default app
WORKDIR - Work folder
CMD - CMD folder
```
